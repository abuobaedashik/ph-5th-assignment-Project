1)What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of an HTML or XML document as a tree of objects, allowing scripts to interact with and manipulate its content and structure.

2)How do you select an element from the DOM?

I can select an element from the DOM using methods like document.getElementById(), document.querySelector(),document.getElementsByClassName()

3)What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to manage events from its child elements, improving performance and simplifying code.

4)How do you manipulate an element's attributes and styles using the DOM?

I can manipulate an element's attributes using setAttribute(), getAttribute() and modify styles using the style property in the DOM
